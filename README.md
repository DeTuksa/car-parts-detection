# Car Parts Detection Model

## Project Structure

dissertation/
├── dataset/
│   ├── Interal/        -Images of internal car parts
│   ├── External/       -Images of external car parts
├── notebooks/          -Jupyter notebooks for data analysis and modeling
├── scripts/            -Python scripts for loading and processing data
├── README.md           -Project documentation
├── requirements.txt    -List of dependencies

## Setup

1. Create a virtual environment
```bash
python -m venv venv
.\venv\Scripts\activate
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Launch Jupyter
```bash
jupyter notebook
```
